### color
palette를 color로 prefix를 정의해 기본 tailwind 값을 overwrite하였고
text 색상은 color-text로 prefix를 정의했습니다.

```css

:root {
  --palette-primary: #4880ee;
  --text-primary: #353c49;
}

@theme inline {
  --color-primary: var(--palette-primary);
  --color-text-primary: var(--text-primary);
}

```

## 개선 포인트
- icon은 svgr을 이용해 react component로 변환하면 좋습니다.
- react component로 변환 시 prop으로 색상 정보를 전달받아, 필요 시 stroke, fill과 같은 svg 스타일 요소들을 커스터마이징할 수 있습니다.
- spacing, radius, ... 여러 디자인 시스템들을 디자이너와 토큰으로 정의하고 tailwind로 작성하면 재사용에 용이합니다.
