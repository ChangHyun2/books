어플리케이션 외부 세계를 infra로 네이밍했고, application에 books repo 계약을 작성한 뒤 infra에서 repo를 구현했습니다.

### infra

infra는 외부 세계이며 http 프로토콜 폴더는 4개의 파일을 포함합니다.

1. dto : 서버 domain to object
2. fetch : api fetch 함수 (ky 라이브러리 사용)
- 요청 값을 검증하기 위해 zod를 사용했습니다.
3. mapper : dto를 fe model로 매핑하는 함수
4. repo : application repo 계약의 구현

### application

application 폴더는 외부 세계와의 경계로, fe 도메인에 의존합니다.
books repo의 계약을 명시했습니다.
